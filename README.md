Web Programiranje Projekat

Ovaj projekat je napravljen kako bi korisnicima omogućio interaktivni rad unutar web stranice, sa posebnim fokusom na tri glavne sekcije: Kanban Board, Vision Board i Whiteboard. Ideja je da korisnici mogu crtati, zapisivati ideje, organizovati zadatke i vizualizirati svoje planove direktno u web pregledniku, bez potrebe za dodatnim programima ili instalacijama. Dodatno, aplikacija omogućava spremanje sadržaja u PDF format, što olakšava čuvanje rada, dijeljenje s drugima i praćenje napretka kroz vrijeme.

O projektu:
Cilj ovog projekta je praktična demonstracija osnovnih web tehnologija (HTML, CSS i JavaScript) kroz razvoj interaktivne web stranice. 
Ideja je pokazati kako se različite sekcije mogu dinamički prikazivati i koristiti bez potrebe za osvježavanjem stranice, što omogućava fluidno i intuitivno korisničko iskustvo.
Projekt omogućava jednostavnu i preglednu navigaciju – korisnik može klikom na meni otvoriti različite sekcije stranice kao što su Whiteboard, Vision Board ili Kanban Board. 
Pored toga, postoji funkcionalnost spremanja u PDF, koja omogućava da sadržaj bilo koje sekcije korisnik preuzme i sačuva, što je praktično za čuvanje bilješki, planova ili zadataka.
Ovaj projekat je takođe koristan za demonstraciju osnovnih principa web programiranja, uključujući rad s HTML-om za strukturu, CSS-om za vizualni izgled i JavaScript-om za dinamičku interaktivnost i manipulaciju sadržajem.

Tehnologije:
Projekt je razvijen korištenjem:

HTML – za izgradnju strukture stranice i organizaciju sekcija
CSS – za dizajn, boje, raspored elemenata i vizualnu estetiku
JavaScript – za dodavanje interaktivnosti, upravljanje sekcijama, crtanje na Whiteboard-u i generisanje PDF-a

Kako pokrenuti projekt:

Pronađi folder projekta na svom računaru, folder u kojem se nalaze index.html, style.css, script.js i svi ostali fajlovi.
Otvori datoteku index.html dvostrukim klikom. Ona će se automatski otvoriti u web pregledniku (npr. Chrome ili Edge).

Funkcionalnosti:

-Početna stranica

Prikazuje osnovne informacije o aplikaciji i uvod u sve funkcionalnosti
Omogućava korisniku da se brzo snađe i odabere željenu sekciju iz menija

-Whiteboard

Otvara se ispod glavnog menija kada se odabere opcija “Whiteboard”
Omogućava crtanje i pisanje
Sve promjene se odmah prikazuju na stranici, bez refresha

-Vision Board

Sekcija za vizualno organiziranje ideja, ciljeva i inspiracije
Korisnik može dodavati slike, tekstualne bilješke i stickere
Pomaže u planiranju i praćenju kratkoročnih i dugoročnih ciljeva na jednom mjestu

-Kanban Board

Omogućava upravljanje zadacima koristeći Kanban metodologiju
Zadaci se mogu dodavati u kolone “To Do”, “In Progress” i “Done”
Pruža jasan pregled napretka, olakšava planiranje i bolju organizaciju aktivnosti

-Snimi kao PDF

Dugme za preuzimanje trenutnog sadržaja aktivne sekcije (Whiteboard, Vision Board, Kanban Board) u PDF format
Omogućava korisnicima da sačuvaju svoj rad i dijele ga s drugima
